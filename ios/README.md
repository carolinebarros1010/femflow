# iOS delivery

This directory centralizes iOS-related packaging guidance for MyFlowLife products.

## Structure
- femflow/ - iOS setup notes for the FemFlow web app
- maleflow/ - iOS setup notes for the Maleflow web app

## Approach options
1) PWA (Safari Add to Home Screen)
2) Native wrapper (WKWebView/Capacitor) when App Store distribution is required

Each product folder contains a checklist and placeholders for assets or future native projects.
