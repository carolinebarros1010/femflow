# lead_data_nascimento

## Resumo
Chave de estado `lead_data_nascimento` usada pelo app FemFlow para controle de fluxo, sessão e experiência da usuária.

## Valor de exemplo
```txt
2026-02-03
```

## Categoria
- Lead / cadastro

## Uso típico
- Lida/escrita no `localStorage` no front-end para persistência entre telas e recargas.

## Sensibilidade
- Não crítico; ainda assim evite exposição desnecessária em logs públicos.
