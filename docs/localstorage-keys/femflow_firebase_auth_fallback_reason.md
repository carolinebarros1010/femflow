# femflow_firebase_auth_fallback_reason

## Resumo
Chave de estado `femflow_firebase_auth_fallback_reason` usada pelo app FemFlow para controle de fluxo, sessão e experiência da usuária.

## Valor de exemplo
```txt
Erro ao sincronizar credenciais com Firebase.
```

## Categoria
- Autenticação / sessão / segurança

## Uso típico
- Lida/escrita no `localStorage` no front-end para persistência entre telas e recargas.

## Sensibilidade
- **Sim** — contém dado sensível; mascarar em logs e nunca expor em prints públicos.
