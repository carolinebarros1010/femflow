# femflow_session_expira

## Resumo
Chave de estado `femflow_session_expira` usada pelo app FemFlow para controle de fluxo, sessão e experiência da usuária.

## Valor de exemplo
```txt
2026-03-19T09:52:55.271Z
```

## Categoria
- Autenticação / sessão / segurança

## Uso típico
- Lida/escrita no `localStorage` no front-end para persistência entre telas e recargas.

## Sensibilidade
- **Sim** — contém dado sensível; mascarar em logs e nunca expor em prints públicos.
