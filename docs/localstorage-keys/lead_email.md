# lead_email

## Resumo
Chave de estado `lead_email` usada pelo app FemFlow para controle de fluxo, sessão e experiência da usuária.

## Valor de exemplo
```txt
intermediario@gmail.com
```

## Categoria
- Lead / cadastro

## Uso típico
- Lida/escrita no `localStorage` no front-end para persistência entre telas e recargas.

## Sensibilidade
- **Sim** — contém dado sensível; mascarar em logs e nunca expor em prints públicos.
