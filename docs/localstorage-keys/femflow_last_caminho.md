# femflow_last_caminho

## Resumo
Chave de estado `femflow_last_caminho` usada pelo app FemFlow para controle de fluxo, sessão e experiência da usuária.

## Valor de exemplo
```txt
{"faseMetodo":"ovulatory","caminho":2,"updatedAt":1771322185465}
```

## Categoria
- Perfil / estado geral

## Uso típico
- Lida/escrita no `localStorage` no front-end para persistência entre telas e recargas.

## Sensibilidade
- Não crítico; ainda assim evite exposição desnecessária em logs públicos.
