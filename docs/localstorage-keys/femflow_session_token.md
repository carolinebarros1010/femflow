# femflow_session_token

## Resumo
Chave de estado `femflow_session_token` usada pelo app FemFlow para controle de fluxo, sessão e experiência da usuária.

## Valor de exemplo
```txt
f2e94744-d56b-4921-a02c-0283e8df1fd5
```

## Categoria
- Autenticação / sessão / segurança

## Uso típico
- Lida/escrita no `localStorage` no front-end para persistência entre telas e recargas.

## Sensibilidade
- **Sim** — contém dado sensível; mascarar em logs e nunca expor em prints públicos.
