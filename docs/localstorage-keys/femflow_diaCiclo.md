# femflow_diaCiclo

## Resumo
Chave de estado `femflow_diaCiclo` usada pelo app FemFlow para controle de fluxo, sessão e experiência da usuária.

## Valor de exemplo
```txt
15
```

## Categoria
- Ciclo hormonal / periodização

## Uso típico
- Lida/escrita no `localStorage` no front-end para persistência entre telas e recargas.

## Sensibilidade
- Não crítico; ainda assim evite exposição desnecessária em logs públicos.
